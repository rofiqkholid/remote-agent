# Agent Monitoring - Quick Start

## Installation

1. **Download**: `AgentMonitoring.exe`
2. **Run**: Double-click the executable
3. **Setup**: Enter server URL: `https://remote.dyanaf.com`
4. **Done**: Agent runs in background (blue icon in system tray)

## Verification

- Open `https://remote.dyanaf.com` in browser
- Your computer should appear in the list
- Click "View" to see your screen in real-time
